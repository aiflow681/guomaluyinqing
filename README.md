【游戏名称】
海豚穿越海洋（重构优化版）

【游戏介绍】
这是一款基于Three.js开发的3D海洋穿越类小游戏。基于原来游戏玩法完全重构。游戏中，你将扮演一只勇敢的海豚，在精美的深海场景中游泳前进，通过敏捷的动作躲避鲨鱼、鱼群和珊瑚礁障碍，挑战更高的分数！

【玩法说明】
海豚需要不断向前推进，穿越不同区域的海洋环境。

障碍物：
1. 鱼群障碍（SmallFishObstacle）：多种色彩的小鱼快速游动，考验玩家的反应。
2. 鲨鱼威胁（SharkObstacle）：体型庞大的食人鲨穿梭于深海，移动速度多变。
3. 海底珊瑚（CoralObstacle）：固定在海床区域的彩色珊瑚，阻挡海豚路径。

操作方式：
- 桌面端：支持键盘方向键 ↑↓←→ 操作，响应迅速。
- 移动端：配备现代化的透明悬浮控制面板，点击按钮即可精准控制。

核心目标：
避开所有活动的生物和静止的障碍，争取游出海洋最远距离。

【技术介绍】
- 引擎：Three.js 3D 渲染引擎
- 架构：ES6 类模块化设计（SceneManager, GameController, GameState）
- 渲染：采用 WebGL 技术，包含实时光照系统、柔和投影及深海渐变背景。
- 优化：集中式 CONFIG 配置管理，解耦了逻辑处理、场景构建与输入反馈。
