【游戏名称】
海豚穿越海洋

【游戏介绍】
这是一款基于Three.js开发的3D海洋穿越类小游戏。游戏中，你扮演一只勇敢的海豚，在广阔的海洋中游泳前进。你要做的就是躲避危险的鲨鱼和小鱼，避开珊瑚礁障碍，游得更远！

【玩法说明】
游戏中海豚需要不断向前游泳，穿越一条条海洋泳道。

障碍物：
泳道上会出现不同的危险和障碍物。
1. 小鱼：橙色、红色、粉色、青色的小鱼在泳道中游动，碰到会游戏结束。
2. 鲨鱼：灰色的大鲨鱼体型更大，移动速度不同，碰到会游戏结束。
3. 珊瑚礁：粉色、橙色、紫色的珊瑚固定在珊瑚礁区域，阻挡你的去路。

操作：
桌面端：使用键盘方向键 ↑↓←→ 控制海豚移动，或点击屏幕按钮。
移动端：点击屏幕上的方向按钮 ↑↓←→ 控制海豚游动。

目标：
碰到任何鱼类或珊瑚就会游戏结束，不要被撞到！

Tips：
游戏会记录你游过的距离（显示在右上角）。
点击"再来一次"按钮，挑战更远的游泳距离记录！

【技术介绍】
本游戏使用Three.js (v0.92) 3D图形渲染引擎开发，主要语言是JavaScript。使用了WebGL渲染技术、正交相机系统、3D几何体建模、Phong/Lambert材质系统、实时光照和阴影等技术。通过BoxBufferGeometry组合构建海豚、鱼类、鲨鱼和珊瑚等3D模型，使用HemisphereLight和DirectionalLight营造海洋光照效果，PCFSoftShadowMap实现柔和阴影，打造出沉浸式的3D海洋穿越体验。
